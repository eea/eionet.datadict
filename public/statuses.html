Registration statuses
 
Each data definition in Data Dictionary (DD) goes through a life-cycle. Its phases are marked by Registration Statuses. They are specified by the data definers and a definition can be in one of the following statuses:
 
Incomplete
Existing dataflows under consideration to be included in DD; new dataflows under development where DD is used as a structuring tool; test dataflows. (At least metadata attributes of Submitter organisation and Submitter phone number are required for this status!).
This status is not published for countries!
 
Candidate
Dataflows to be included in DD (of relevance to countries). This level is for opening discussions between ETCs and EEA. (At least metadata attributes of Submitter organisation and Submitter phone number are required for this status!).
This status is not published for countries!
 
Recorded
All mandatory metadata entered! This level is for discussions between ETCs and countries. So it is published for countries as well!
 
Qualified
All mandatory metadata entered! This level is for closing discussions between ETCs and EEA. Not published for countries!
 
Released
All mandatory metadata entered! Accepted by ETCs and EEA. Released for countries to use for download of technical guidelines and templates.
Naturally, it is published for countries too!
