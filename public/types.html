Data Dictionary



 
EIONET Data Dictionary Table & element types
 
 
Data elememt types:
Elements with fixed values (codes) - data elements with values from a pre-fixed set
Elements with measured values - aka elements with quantitative values or simply measurements.
 
Table types:
Normal - tables that do not participate in any foreign key relationships with any other tables.
Lookup - tables that participate on the "many" side of the one-to-many foreign key relationships between tables in a single dataset. Hence the name lookup tables.
