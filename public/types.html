Table & element types

Data element types:

Elements with fixed values (codes) - Data elements with values from a pre-fixed set.

Elements with measured values - Elements with quantitative values or simply measurements.

Table types:

Normal - Tables that do not participate in any foreign key relationships with any other tables.

Lookup - Code lists or fixed value lists or suggested values. These are tables that participate on the "many" side of the one-to-many foreign key relationships between tables in a single dataset. Hence the name lookup tables.
